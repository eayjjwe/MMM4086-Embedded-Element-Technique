# MMM4086-Embedded-Element-Technique
Script developed as part of the University of Nottingham's MMME4086 module in the final year MEng Mechanical Engineering programme.

GNU GPLv3 license applies.

All versions are a WiP and may possess faults throughout.
The version titled "w/out ori" ignores the prescribed local orientation system and applied a generic global one. In effect turning the fibre material isotropic.
The version titled "w/ ori" includes the local orientation system, but as of 05/23 presents fatal errors when the jobs are run in ABAQUS Command.
